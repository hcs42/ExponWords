Version 0.14 (2013-08-11)

New in this release:

  • New, smarter algorithm for strengthening words during practice. The old algorithm can still be used, but all existing and new dictionaries use the new algorithm by default. The old and new algorithms and the difference between them is described in the Methods of strengthening a word section of the help.
  • Practice early: words can be practiced even if there are no words due today (by using the new "Practice early" link on a dictionary's page).
  • Words and notes in the dictionaries can be formatted using the HTML language. For example parts of the words can be mode bold, italic, and images can be inserted. The recommended way of starting using HTML formatting on an existing dictionary is to set the format of the dictionary to "HTML (keep line breaks)" in the "Modify dictionary" page, and then just start using HTML tags in the words and notes. More information and examples can be found in the Text formats section of the help.
  • The buttons on the Practice page can be controlled with a presenter device (the PageUp/PageDown settings should be changed in Settings to enable this).
  • The practice page has a full screen mode.
  • The practice page has more keyboard shortcuts: M for showing the operations, and F for full screen mode.

Version 0.13 (2012-12-10)

New in this release:

  • Word pairs can be deleted from the "Edit word pair" page.
  • When a new word pair is added via the "Add new word pair" page and a similar word pair has already been added to the same dictionary, the user will be warned. Two word pairs are considered similar if at least of their words is the same.
  • The "Practice page" shows quick labels (after clicking on the "More" button), which can be added to the current word quickly. The list of quick labels can be set in Settings; by default only the label called "quick" is a quick label.
  • The "Practice page" will reload itself automatically if there are still words to be practiced.
  • The "Practice page" will only load a maximum of 200 words at a time.
  • The word orderings in the "Practice page" are calculated using tomorrow's dimness instead of today's dimness.

Version 0.12 (2012-06-10)

New in this release:

  • The user can choose from different word orderings on the "Practice page". This can be set globally in the "Settings" page and separately for each dictionary in the "Modify dictionary" page.
  • The "Add new word pair" and "Edit word pair" pages have a basic and an advanced view.
  • The "Add new word pair", "Edit word pair" and "Search" pages have a new form for changing dates and strengths.
  • Help pages are opened in new tabs.
  • Password-related pages look like other ExponWords pages.
  • New users will get a welcome email.
  • New users can opt out from receiving release emails on the registration form.

Version 0.11 (2012-02-04)

New in this release:

  • Words can have multiple lines.
  • The "explanation" field of the word pairs was renamed to "notes".
  • The "Search and operations" page divides the result into several pages.
  • Word pairs selected on the "Search and operations" page can be exported.
  • The "Practice" page shows the labels when showing the answer.
  • Links to editing the word pairs on the "Practice" page are hidden behind a "More" button.

Version 0.10 (2011-11-06)

New in this release:

  • New web address: exponwords.com.
  • The names and languages of dictionaries can be modified.
  • Links on the front page to the "Practice" pages. The link text is the number of words due today.
  • The "Practice" page shows the direction of the current question by printing an arrow.
  • The front page shows the list of dictionaries in an alphabetical order.

Version 0.9 (2011-09-15)

New in this release:

  • Some things were renamed: "Action" → "operation", "Search" page → "Search and operations" page, "Options" → "Settings".
  • The word pairs on the "Search and operations" page can be selected not only using the checkboxes, but by clicking anywhere on the word pair's row. The selected lines become gray.
  • After adding a word pair, the next page will contain a link to the previously added word pair.
  • You can register an account, change your password and reset it in case you have forgotten it.
  • On the "Settings" page, the time zone and the turning point can be set besides the language. (The turning point is the time of the day when ExponWords will start asking the words of that day.)
  • You can set your email address and whether you would like to receive emails about new features.
  • You can choose between two arrangements for the "Practice" page and you can set the size of the buttons and text too.

Version 0.8 (2011-08-28)

New in this release:

  • The "Practice" page can be used with keyboard shortcuts:
    • pressing any key is equivalent to clicking on the "ANSWER" button;
    • pressing the "Y" or "I" key is equivalent to clicking on the "YES" button;
    • pressing the "N" key is equivalent to clicking on the "NO" button.
  • Listing the hits can be turned off on the "Search" page. This is useful if you want to do the same operation on hundreds of word pairs (e.g. shift their dates), but the web browser would be too slow because of the large page or you want to spare your network usage.
  • Labels can be specified when importing word pairs; these labels will be added to the words.
  • The "Show my future" page (available from the front page) was created that draws a graph displaying a prediction about the number of words that will be asked each day in the next two months.

Version 0.7 (2011-08-10)

New in this release:

  • The word pairs selected in the "Search" page can be practiced. You can choose either practicing all the selected words, or only those that are due.

Version 0.6 (2011-07-30)

New in this release:

  • On the "Search" page, the dates and strength of the selected word pairs can be modified at once, and also labels can be added, removed and modified at once.
  • When a new word pair is added, ExponWords remembers the initial dates, strengths and labels. This way if several word pairs are to be added e.g. with the same label, the label has to be typed only for the first word pair; it will already be present in the form for the rest of the word pairs.

Version 0.5 (2011-07-11)

New in this release:

  • Moving word pairs (available from "Search page"): Multiple word pairs can be moved to another dictionary.
  • Shift dates (available from "Search page"): The dates of the word pairs (based on which the program decides when to ask the word pair) can be shifted with the number of given days. For example if 3 days is typed, then a word which would otherwise be asked on 1st January will be asked only on 4th January. Negative numbers can also be specified: in this case the dates will be changed to earlier ones. This can be useful if you did not have time to practice for a week, but do not want all the words due to be asked on the first day. Using this feature, ExponWords can act as if that one week did not exists.

Version 0.4 (2011-06-27)

New in this release:

  • Labels can be assigned to word pairs when editing them (a word pair may have several labels). They can be used to organize the dictionaries.
  • The new "Search" page (which can be opened either from the main page or the pages of the dictionaries) allows us to search among the word pairs. The dictionary and the label to be searched can be selected.
  • Negative strengths can be used. For example if -2 is set, then the word will be asked today, tomorrow, the day after tomorrow, and only then will the frequency decrease.

Version 0.3 (2011-03-20)

New in this release:

  • Code released under GPLv3 (as open source software).
  • "(Help)" links on more pages.
  • "Select all" button added to the "View" page.
  • Better user experience during practice sessions with bad network connections (the timeouts in the client were increased).
  • FAQ translated to English help pages. (It was already present on the Hungarian help page.)

Version 0.2 (2011-03-13)

New in this release:

  • ExponWords rewritten using the Django web framework.
  • Users can log in using usernames and passwords.
  • You can create multiple dictionaries.
  • All details of a word pair (dates of next practices, strengths, etc.) can be changed.
  • Word pairs can be deleted.
  • The date of addition is stored for each word pair.
  • The "Practice" page displays a link to the previous word (it can be useful if you realize that you did not actually know a word just after pressing "YES"; this way you can easy edit the word's strength).
  • Better user interface, e.g. menus.

Version 0.1 (2011-02-13)

The first release of ExponWords had the following features:

  • Word pairs can be added and edited.
  • "Practice" page with link to editing the words.
  • Word pairs can be copy-pasted from a spreadsheet application and added at once.
  • English and Hungarian user interface and help pages.
  • No support yet for multiple dictionaries and multiple users.
